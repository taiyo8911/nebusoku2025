# nebusoku2025

## プロジェクト概要
「Lack of Sleepと寝不足にならNight 10」のイベントサイト

### 開催情報
- **イベント名**: Lack of Sleepと寝不足にならNight 10
- **主催**: Lack of Sleep
- **日程**: 9月29日(月)・30日(火)
- **時間**: 両日 16:20〜20:20
- **会場**: LIVE HOUSE ANGA
- **料金**: 1,000円（1ドリンク付）、高校生以下無料

## 技術仕様

### HTML
- セマンティックなマークアップ
- OGP設定（title、description、type、locale）
- レスポンシブ対応のメタタグ
- アクセシビリティ対応（適切なalt属性、ARIA）
- 外部リンクのセキュリティ対応（rel="noopener"）

### CSS
- CSS Grid Layout（デスクトップ: 3カラム、モバイル: 1カラム）
- CSS Custom Properties（カラーパレット管理）
- レスポンシブデザイン（4段階のブレークポイント）
- ダークテーマ
- グラデーション効果
- スティッキーポジショニング
- カスタムスクロールバー

### JavaScript
- 軽量設計（外部ライブラリ依存なし）
- Intersection Observer API（統計カウンター）
- パフォーマンス最適化（requestAnimationFrame使用）
- アクセシビリティ機能（キーボードナビゲーション）

## レスポンシブデザイン

### ブレークポイント
- **デスクトップ**: 1201px以上（3カラムレイアウト）
- **タブレット**: 769px-1200px（1カラムレイアウト）
- **モバイル**: 481px-768px（1カラムレイアウト、調整済み）
- **スマートフォン**: 480px以下（最小サイズ対応）

### レイアウト変更点
- **1200px以下**: 3カラム → 1カラムに変更
- **表示順序**: ヘッダー → ヘッドライナー → プログラム → 統計 → フッター
- **スティッキー要素**: モバイルでは無効化

## カスタマイズ

### 色彩変更
`css/style.css`の`:root`セレクター内のCSS変数を変更

### アニメーション調整
`js/animation.js`内の以下の値を変更：
- フェードイン間隔: `i * 150`（ミリ秒）
- カウンター速度: `duration = 1000`（ミリ秒）
- イージング: `1 - Math.pow(1 - progress, 3)`

### レスポンシブ調整
メディアクエリのブレークポイントを変更：
- `@media (max-width: 1200px)`: タブレット対応
- `@media (max-width: 768px)`: モバイル対応
- `@media (max-width: 480px)`: スマートフォン対応
